# EnergyMe Examples & Integrations

Community-contributed scripts, integrations, and examples for EnergyMe devices.

## Available Examples

- **[Multiple Device Dashboard](multiple-energyme/php-ajax/)** - PHP/AJAX dashboard to aggregate multiple EnergyMe devices

## Contributing

Have a useful integration? Submit a pull request!

1. Create a folder: `examples/<category>/<your-example>/`
2. Include a README.md with setup instructions
3. Add your code/configuration files

## Ideas Welcome

- Home Assistant integrations
- Node-RED flows
- Python scripts (logging, analysis)
- Grafana dashboards
- MQTT clients
- Mobile apps

Check the main project documentation for API details: `http://energyme.local/api/v1/`
